Mobile Header Menu

Your task will be to create mobile menu, common called as 'hamburger menu'.

This menu has already created HTML construction. Every one of you have to add JS and CSS code to make this menu functional.

You can change HTML if you think this is necessary. Also, you can use any tool or plugin.

Rules:

  1. Menu is accessable from red button placed in header.
  2. Header is fixed and have to always visible. When menu is opened, it should bo below header.
  3. Menu should have fixed position too. You could open it from any position (scroll position) of page.
  4. If menu is shorter than screen height, rest of screen should have black background with 80% of transparency.
  5. Menu has 2 levels of nesting. Some elements have own submenus. Only one submenu can be visibe. Opening one of submenus should close rest submenus.
  6. If menu is heigher than window height, user should have possibility to scroll it. Menu should go under header if this is necessary.
  7. If menu is opened, user have to have blocked scrolling of rest page. Only element of menu can be scrolled if it's opened.

If something is not clear, please let us know. We will clarify everything! :)
